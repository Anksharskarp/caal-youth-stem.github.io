Helpful Hands

In this project we used our computer science skills to help create a website for the youth community service program, Helpful Hands. Our easy-to-navigate and eye-catching website provides a centralized website for a user to view Helpful Hands' past and future projects, read about stories of youth volunteers, view announcements, and provide ways for others to support this group.
